Intro

I decided to put this together to show and tell about my professional abilites. It was written completely by hand, using little more than a text editor with StackOverflow and reference docs at the ready. The stylesheet is written in Sass using the Bourbon library and Bourbon's Neat framework. I also used the command line to compile the sass files and used GitHub for version control. Finally, I set up my local environment using Vagrant. The javascript is written by hand, and while I used several sites as inspiration/troubleshooting, everything has been customized by me to fit my needs.

I'm a web developer with a passion for improving the user's experience. I'm fluent in HTML and CSS, prefer semantic markup and can work well as an individual or part of a team. I've actually never used Bootstrap or Foundation CSS frameworks because I feel like it can often add unnecessary bloat to a project and can often muddy the markup. While those frameworks can allow for rapid development, I prefer to write my own so that I have more control over the output without having to overwrite a framework's styles.

I've divided this resume into six parts: Intro, Relevant Experience, Other Jobs, Portfolio, Education and Contact. I hope this works to sell you on me. And if you have any questions, please let me know.
