Intro

This resume is in beta*. I decided to put this together to show and tell about my professional abilites. It was written completely by hand, using little more than a text editor (Atom) with (lots of) StackOverflow and reference docs at the ready. The stylesheet is written in Sass using the Bourbon library and Bourbon's Neat framework. It is compiled with Grunt. The javascript is written by hand, and while I used several sites as inspiration/troubleshooting, everything has been customized by me to fit my needs.

I'm a web developer with a passion for improving the user's experience. I'm fluent in HTML and CSS, prefer semantic markup and can work well as an individual or part of a team. I've actually never used Bootstrap or Foundation CSS frameworks because I feel like it can often add unnecessary bloat to a project and can often muddy the markup. While those frameworks can allow for rapid development, I prefer to write my own so that I have more control over the output without having to overwrite a framework's styles.

I've divided this resume into four parts: Intro, Relevant Experience, Other Jobs, and Contact. I hope this works to sell you on me. And if you have any questions, please let me know.

*I mention that this resume is in beta for a few reasons. I've tested it across multiple desktop browsers and mobile devices and it generally performs well but is not without its bugs. If you find them, please forgive me. If you don't, you should definitely hire me!
